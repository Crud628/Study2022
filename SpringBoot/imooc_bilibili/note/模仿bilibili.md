## 1.1  项目架构

### 1.1.1  项目架构分类

#### 1.1.1.1  业务（功能）架构

顶层：用户服务，如注册登录、大会员权限、查找感兴趣视频等

中间层：在线视频流播放 + 实时弹幕

底层：管理后台，如视频上传、数据统计、系统推送

#### 1.1.1.2  技术架构

技术选型：SpringBoot2.x + MySQL + MyBatis + Maven

开发模式：MVC

#### 1.1.1.2  部署架构

前端：服务转发 + 负载均衡

后端：业务处理 + 功能实现

工具：缓存、队列
